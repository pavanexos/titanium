# GlassSuite (Vite + React + TS + Tailwind + Radix)

## Run locally
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

This template uses:
- Vite + React 18 + TypeScript
- TailwindCSS
- Radix UI (accessible primitives)
- shadcn-style UI wrappers in `src/components/ui/*`
